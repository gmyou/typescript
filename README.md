# typescript

## Installation

```bash
npm init
npm install typescript --save
./node_module/.bin/tsc --init
```
or
```bash
npm init
npm install -g typescript
tsc --init
```

### extension
tslint for Code convention
```bash
npm install tslint --save
./node_module/.bin/tslint --init
```
or
```bash
npm install -g tslint
tslint --init
```
